The Model — Motivation, Parameters & Validity

Why this equation?

Capital in financial markets behaves like a physical field: it diffuses across assets, has inertia (trends), and is pulled toward fundamental equilibrium. Classical Ornstein-Uhlenbeck (O-U) models capture mean reversion but ignore (1) network structure, (2) varying diffusion reach across regimes, and (3) real-time feedback from prediction error. This model extends O-U to a graph field equation where each asset is a node with a capital density mi(t):
γ(t)·m''i + m'i = −α(t)·Lzs(t)·mi + fiBayes(t) + Ωi(t) + v(t)
fiBayes(t) = fifund(t) + κ · δi(t) · Sfear(t)   where   δi(t) = mpredi − mreali

Validity analysis

The equation is physically motivated but simplified. Key assumptions and their limits:
  • Linear mean-reversion: the −α·L^s·m term assumes capital returns to equilibrium proportionally to displacement. This holds in normal regimes but breaks down in structural breaks (e.g., sector rotation, regulatory changes). The Bayesian δ-feedback partially compensates by re-anchoring the source term.
  • Static graph topology: edges (correlations) are recalculated daily but the zone structure (USD / EUR / ASIA / EM) is fixed. Cross-zone capital flows via bank nodes only — a reasonable approximation given Basel III constraints, but underestimates shadow-banking channels.
  • Fractional Laplacian L^s: the exponent s controls diffusion reach. s=1 recovers standard graph diffusion. s→0 makes every node directly connected to every other (fully non-local). The heuristic mapping VIX→s is calibrated empirically; a more principled approach would infer s from realized correlation matrices.
  • Bayesian κ=0.05: conservative by design. A higher κ closes the feedback loop faster but risks instability when δ is noisy (e.g., data latency). The S_fear dampening ensures κ·S_fear → ~0.03 in panic, preventing overcorrection precisely when data quality degrades.

How to interpret the results

A __WIN_RATE__% win rate on 5-day forward returns is measured in-sample on the same 90-day window used to generate signals — this is an exploration metric, not a validated backtest. Signals are filtered by p_reversion > 0.55, composite score threshold, and reversibility check (P7). The model is not a price predictor — it is a regime-aware mean-reversion detector. Its edge comes from correctly identifying when O-U dynamics dominate (calm/normal regimes, VIX 15–22). Disclaimer: α, γ, s, S_fear in this demo are derived from a VIX→regime lookup table, not from running the actual EKF/UKF filters. The chart below shows the lookup values, not Kalman posterior estimates.

Regime Parameter Evolution — α, γ, s, S_fear + VIX (second axis)

How to read this chart: When VIX spikes, α increases (faster diffusion — capital flows spread across more assets simultaneously), γ collapses (momentum vanishes — no trend-following in panic), s drops (global contagion — the fractional Laplacian reaches across zones instead of staying local), and S_fear dampens signals — the model simultaneously becomes more cautious and faster-reacting. In calm regimes (VIX < 15), high γ and s near 0.9 mean slow diffusion and persistent trends.
